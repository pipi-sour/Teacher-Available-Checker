Accessible, screen reader friendly, MDL buttons

To try out the screen reader friendly MDL buttons you must have a screen reader plugin installed, e.g. Chrome Vox.

A plain MDL button is accessible

<button class="mdl-button mdl-js-button mdl-button--raised">
  Add item
</button>
OK. Screen readers reads the text in the button tag.

The MDL FAB button is not accessible

<button class="mdl-button mdl-js-button mdl-button--fab
        mdl-button--colored">
  <i class="material-icons">add_shopping_cart</i>
</button>
NOT OK. Screen readers reads the text in the <i> tag.

Make the MDL FAB button accessible

1. Hide text content in <i> tag from screen readers using aria-hidden="true". 
<button class="mdl-button mdl-js-button mdl-button--fab
        mdl-button--colored">
  <i class="material-icons" aria-hidden="true">
    add_shopping_cart
  </i>
</button>
The text in the <i> tag is hidden from screen readers. Screen readers read "button".
... or use a CSS class for the Material Icon name. 
.add-shopping-chart::after {
  content: 'add_shopping_cart';
}

<button class="mdl-button mdl-js-button mdl-button--fab
        mdl-button--colored">
  <i class="material-icons add-shopping-chart"></i>
</button>
The text in the <i> tag is hidden from screen readers. Screen readers read "button".
2. Use the aria-label attribute for the button text. 
<button class="mdl-button mdl-js-button mdl-button--fab
        mdl-button--colored aria-label="Add item to cart">
  <i class="material-icons add-shopping-chart"></i>
</button>
Screen readers reads the aria-label text.
... or add text content to FAB button - but hide the content from visual representation. 
.visually-hidden {
  border: 0;
  clip: rect(0 0 0 0);
  height: 1px;
  margin: -1px;
  overflow: hidden;
  padding: 0;
  position: absolute;
}

<button class="mdl-button mdl-js-button mdl-button--fab
        mdl-button--colored">
  <span class="visually-hidden">Add item to cart</span>
  <i class="material-icons add-shopping-chart"></i>
</button>
Screen readers reads the <span> text content.

Make the MDL icon button accessible

<button class="mdl-button mdl-js-button mdl-button--icon"
        aria-label="Listen to radio">
  <i class="material-icons" aria-hidden="true">radio</i>
</button>
Screen readers reads the aria-label text. 
<button class="mdl-button mdl-js-button mdl-button--icon">
  <span class="visually-hidden">Listen to radio<span>
  <i class="material-icons" aria-hidden="true">radio</i>
</button>
Screen readers reads the <span> text content.

Accessible MDL button decorated with icon

<button class="mdl-button mdl-js-button mdl-button--raised">
  <i class="material-icons" aria-hidden="true">radio</i>
  Listen to radio
</button>
Screen readers reads the text in the button tag.

Accessible MDL button decorated with <img>

<button class="mdl-button mdl-js-button mdl-button--raised">
  <img  src="..." style="height: 70%"
        aria-hidden="true" role="presentation">
  Listen to radio
</button>
Screen readers reads the text in the button tag.
aria-describedby attribute

Click to browse internet radio stations.

<p id="www-radio-description">
  Click to browse internet radio stations.
</p>
<button id="www-radio-button"
        class="mdl-button mdl-js-button mdl-button--raised"
        aria-describedby="www-radio-description">
  <i class="material-icons" aria-hidden="true">radio</i>
  Listen to radio
</button>
Screen readers reads the text in the button tag, then the text in the p tag.
visually-hidden + aria-describedby + title attribute

Click to browse internet radio stations.

<p id="www-radio-description" class="visually-hidden">
  Click to browse internet radio stations.
</p>
<button id="www-radio-button"
        class="mdl-button mdl-js-button mdl-button--raised"
        aria-describedby="www-radio-description"
        title="Click to browse internet radio stations.">
  <i class="material-icons" aria-hidden="true">radio</i>
  Listen to radio
</button>
Screen readers reads the text in the button tag, then the text in the visually hidden p tag. Shows the title attribute value as a “tooltip” on mouseover.
